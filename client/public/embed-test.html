Iframe Embed Test Harness

Validates that /embed/* routes load inside iframes and non-embed routes are blocked.

Testing Method

  1. Open this page from an approved domain (e.g., https://coachstevenbaseball.com) — the embed iframe below should load.
  2. Open this page from an unapproved domain (any other origin) — the embed iframe should be blocked by CSP.
  3. The "Non-Embed Route" test should always fail regardless of origin — the homepage is not embeddable.
